INTERNAL CSS METHOD


(This is styled using INTERNAL CSS METHOD)----Internal CSS in HTML means adding CSS code in the section of the document. Styling changes apply to every specific element found in the file. Tip: this is a solid option for one-page websites or when you cannot create external stylesheets for your project





                                                                                                                                                          INLINE CSS METHOD

(This is styled using INLINE CSS METHOD)----Inline CSS An inline style may be used to apply a unique style for a single element. To use inline styles, add the style attribute to the relevant element. The style attribute can contain any CSS property.
